# Feelin'Food Mobile
###### Designing an application around food delivery 
-----------------------------------------------------------------------


**A Snack Expo project.**

## Install Expo
```
npm add expo
```

## Project setup
```
npm install
```

## Run Expo CLI
```
expo start
```


### Try it ! 
See the [Snack DEMO](https://snack.expo.io/@ypariset/feelinfood-mobile).

Enjoy!
